# 《法学引注手册》格式规范（完整版）

参考《法学引注手册》（2019年版）整理。适用场景：法学论文、课程论文、学术写作中的引注格式统一。

---

## 一、通用规则

### 1.1 标点符号
- 中文文献：全文使用**全角标点**（，。：；！？《》「」）
- 英文文献：使用**半角标点**（, . : ; ! ? " "）
- 作者之间分隔：中文用**顿号** `、`，英文用**逗号+空格** `, `
- 中文文献方括号使用**半角** `[]`（如 `[M]`、`[J]`），但参考文献列表中的序号可使用半角

### 1.2 作者
- 最多列出 3 位作者，超过 3 位的只列前 3 位后加"等"或"et al."
- 中文作者名之间用顿号：`张三、李四：《文章名》`
- 英文作者名之间用逗号+空格：`Smith, John, and Mary Jones, Title`
- 外文作者采用"姓在前，名在后"格式

### 1.3 页码
- 格式：`第XX-XX页` 或 `第XX页`
- 页码范围用半角短横 `-`（不加空格）
- 多个不连续页码用顿号：`第19-52、204-205页`
- 脚注引用页码和参考文献列表页码格式相同
- 引用页用 `第XX页`，引用起止页用 `第XX-XX页`

### 1.4 数字
- 年份、期号、页码用**阿拉伯数字**
- 卷号用阿拉伯数字
- 中文数字（一、二、三）仅用于古籍卷次等特定场合

### 1.5 重复引用
- 同一文献再次引用时，使用：`同上注，第XX页`
- 作者同上、文献不同的，使用：`同上揭` 或作者名+篇名

## 二、各类文献格式

### 2.1 期刊论文

```
作者：《篇名》，载《刊名》年份年第X期，第XX-XX页。
```

**注意：**
- 不标注卷号（"第X卷"），只标年份和期号
- 期刊名加《》，不区分是否连续出版
- 引文页码在前，参考文献页码在后（脚注中两者格式相同）

**示例：**
```
谢元鲁：《论中国古代国家监察制度的历史经验》，载《社会科学辑刊》2004年第3期，第83-88页。
```

**英文期刊：**
```
Author, "Title," Volume Journal Name Page (Year).
```
示例：
```
Charles A. Reich, "The New Property," 73 Yale Law Journal 733 (1964).
```

### 2.2 专著

```
作者：《书名》，出版社年份版，第XX页。
```

**注意：**
- 出版年版指版本年份，如"2019年版"
- 初版不标版次："北京大学出版社2019年版"（不写"第1版"）
- 出版社前不加出版地（不同于 GB/T 7714）

**示例：**
```
王泽鉴：《英美法导论》，北京大学出版社2011年版，第99-100页。
苏力：《法治及其本土资源》，北京大学出版社2015年第3版，第25页。
```

**英文专著：**
```
Author, Title (Publisher Year).
```
示例：
```
Ronald Dworkin, Law's Empire (Harvard University Press 1986).
```

### 2.3 主编/编著作品

```
主编/编者名（主编/编）：《书名》，出版社年份版，第XX页。
```

**示例：**
```
王利明（主编）：《民法》，中国人民大学出版社2018年版，第50页。
```

### 2.4 文集/集刊论文

```
作者：《篇名》，载主编名（主编/编）：《文集名》，出版社年份版，第XX-XX页。
```
或
```
作者：《篇名》，载《集刊名》第X卷/辑，出版社年份版，第XX-XX页。
```

### 2.5 译著

```
[国别]作者名：《书名》，译者名译，出版社年份版，第XX页。
```

**示例：**
```
[德]卡尔·拉伦茨：《法学方法论》，陈爱娥译，商务印书馆2003年版，第30页。
```

### 2.6 学位论文

```
作者：《篇名》，博士/硕士学位论文，学校名称年份年，第XX页。
```

**注意：** 只标注博士论文和硕士论文，不标注学士论文。

**示例：**
```
贺欣：《离婚法院的诉讼过程》，博士学位论文，斯坦福大学2003年，第50页。
```

### 2.7 报纸文章

```
作者：《篇名》，载《报纸名》年月日，第X版。
```

**示例：**
```
孟亚生：《地域史视野下监察文化传承与借鉴》，载《新华日报》2026年5月22日，第12版。
```

### 2.8 网络文献

```
作者：《篇名》，载网站名，URL，最后访问日期。
```

**注意：**
- 标注最后访问日期（年月日）
- 如有发布日期也一并标注
- URL 不需要特别格式（不需尖括号等）

**示例：**
```
苏永钦：《裁判的方法》，载中国法学网，http://www.iolaw.org.cn/showNews.asp?id=1234，2024年3月1日最后访问。
```

### 2.9 法律法规

```
《法规名称》（通过/修订日期）
```

**示例：**
```
《中华人民共和国民法典》（2020年5月28日第十三届全国人民代表大会第三次会议通过）
《中华人民共和国公司法》（2018年修正）
```

### 2.10 司法案例

```
案例名称，案号（裁判日期）
```

**示例：**
```
深圳市快播科技有限公司等传播淫秽物品牟利案，北京市海淀区人民法院（2015）海刑初字第512号刑事判决书（2016年9月13日）。
```

### 2.11 古籍

```
（朝代）作者：《书名》卷第X，《篇名》。
```

**示例：**
```
（清）沈家本：《历代刑法考》卷三，《刑制总考三》。
```

### 2.12 会议论文

```
作者：《篇名》，载会议名称论文集，出版社年份版，第XX-XX页。
```

## 三、参考文献列表专项规则

### 3.1 排序
- 中文文献在前，英文文献在后
- 中文按作者姓氏拼音排序
- 英文按作者姓氏字母排序
- 同一作者多篇文献按出版年份先后排

### 3.2 编号与缩进
- 编号格式：`[1]` `[2]` ...（半角方括号+阿拉伯数字）
- 编号 1-9：悬挂缩进 0.6 cm
- 编号 10-99：悬挂缩进 0.74 cm
- 编号 100 以上：悬挂缩进 0.9 cm

### 3.3 参考文献 vs 脚注的区别
- 脚注：引用具体页码（如 `第83页` 或 `第83-88页`）
- 参考文献列表：期刊标注起止页码，专著可不标页码
- 两者格式基本一致，主要差异在是否需要页码范围

## 四、常见错误

| 错误 | 正确 |
|------|------|
| 作者之间用逗号 `，` | 作者之间用顿号 `、` |
| 页 `25-30` | 页 `第25-30页` |
| `社会科学辑刊，2004年第3期` | `《社会科学辑刊》2004年第3期` |
| 出版社前加出版地 | 不加出版地 |
| `[J].` 期刊标识符保留 | 输出格式中不出现 `[J]` |
| 初版标注"第1版" | 初版不标注版次 |
| 期刊标注卷号 | 不标卷号，只标期号 |

## 五、normalize_citations.py 支持格式

脚本 `scripts/normalize_citations.py` 可将以下输入格式批量转换为前述标准格式：

| 输入来源 | 格式特征 | 示例 |
|----------|---------|------|
| 知网 GB/T 7714 | `作者.篇名[J].刊名,年(期):页码.` | `谢元鲁.论中国古代...[J].社会科学辑刊,2004(03):83-88.` |
| 知网文献节 | `篇名[J].作者.刊名,年(期)` | `论中国古代...[J].谢元鲁.社会科学辑刊,2004(03)` |
| 半成品 | 接近目标，标点或格式有小差别 | `谢元鲁：《论中国古代...》，载《社会科学辑刊》2004年第3期，第83-88页。` |
| 报纸 | 含年月日+版 | `孟亚生：《...》，载《新华日报》2026年5月22日第12版。` |

输出统一为《法学引注手册》格式。
